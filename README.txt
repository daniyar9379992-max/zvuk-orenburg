Как открыть сайт Zvuk

1. Распакуйте архив в любую папку.
2. Откройте файл index.html двойным кликом.
3. Если браузер спросит разрешение на открытие локального файла, разрешите.

Для телефона лучше открыть сайт по интернет-ссылке. Для этого папку сайта можно загрузить на любой статический хостинг, например Netlify Drop или GitHub Pages.
